async_demo.py
-Использует asyncio и TaskGroup.
-Все задачи запускаются конкурентно.
-Общее время выполнения ≈ времени самой долгой задачи.

threading_demo.py
Использует threading.
-Потоки работают конкурентно, но GIL не даёт им реально выполняться параллельно.
-Полезно для IO-операций, но не для тяжёлых вычислений.

multiprocessing_demo.py
-Использует multiprocessing.
-Каждая задача выполняется в отдельном процессе → параллелизм.
-Может задействовать несколько ядер CPU.
